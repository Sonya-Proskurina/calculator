# A simple calculator on flutter

## Functional 

- +, -, *, /
- Trigonometric functions: sin, cos, tg, ctg, asin, acos, atg, act
- e^x, 10^x, x^y
- lnx, lgx
- Constants: pi, e
- Exponential notation of numbers

## Technologies

- flutter_bloc: ^8.0.0
- get_it
